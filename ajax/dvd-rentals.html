DVD Rentals: A SQL Project

Project Info:

This project used a provided dataset for a DVD rental company and the SQL written answers the following business questions, "Which customers have the most rentals?" and "What movies/genres are each customer renting along with the customer's contact information?". These questions will assist the business in determining if they should implement a loyalty program as well as individualize marketing campaigns based on the category of movies rented.

Project Details:

  • Technologies:SQL, PostgreSQL, Import/Export and Transform Data, Create Procedures and Functions, Create Triggers
