Events Project

Project Info:

A Ruby on Rails application that allows users to sign up/sign in to create or register for events. Users are able to like events as well. Users are able to see who is registered for an event as well as which users have liked an event.

Project Details:

  • Technologies:Ruby on Rails, HTML, CSS, SASS, PostgreSQL, Git, Heroku
  • URL:tech-eventz.herokuapp.com
  • GitHub:github.com/davidblackwelder/eventz
