Forest Prime: A/B Test Data Analysis Project

Project Info:

This project analyzed the results of A/B testing for an ad campaign run by the fictional e-commerce company, Forest Prime.

Project Details:

  • Technologies:Anaconda, Jupyter Notebook, Python, Pandas, Matplotlib, Numpy, Seaborn, Statsmodels, Github
  • GitHub:github.com/davidblackwelder/forest-prime-ab-test
