Memory: A Coding Project

Project Info:

This is the project I submitted in the hiring hackathon for Mintbean. The challenge was to build a 2D game with a clear win condition as well as a clear lose condition in 7 days. The frontend had to be written in JavaScript or Typescript. I chose to build a memory game because my daughter loves playing this type of game. I used Fontawesome to pull in chess icons because I enjoy playing chess. I may add more features to this over time such as the ability to save scores and view a leaderboard of player's times.
